The interface for the key-value map internal to an ArenaSet.

The Entry API is not supported, because it can't be used as is, anyway: the reference passed as a key to entry(K) would be to something outside the arena_set, which we absolutely don't want to store in the map. The Entry API would have to be extended to allow changing the key before insertion.
